## :warning: Rodando o projeto
### Crie um ambiente virtual
```
virtualenv myenv 
```
### Ative o ambiente criado
```
myenv\Scripts\Activate
```
### Instale o pacote Django
```
pip install django
```
### Rode o projeto
```
python3 manage.py runserver
```
